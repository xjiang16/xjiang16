### 🚀 Featured Projects

#### 🏙️ [Austin Airbnb Data Pipeline](https://github.com/xjiang16/austin-airbnb-data-pipeline)
**Stack: Python (Pandas), PostgreSQL, SQL**
> Living in the Austin area, I wanted to see the "why" behind local housing trends. I built an end-to-end ETL pipeline that:
> * **Extracts** raw data from Inside Airbnb.
> * **Transforms** 79 messy columns into 18 high-impact features using Pandas.
> * **Loads** the result into a PostgreSQL database for structured SQL analysis.

#### 🧱 [LEGO Portfolio Tracker](https://github.com/xjiang16/lego-portfolio-api)
**Stack: Python, RESTful APIs, JSON**
> I treat LEGO sets like an asset class. This backend API tracks my collection’s ROI, managing price fluctuations and calculating profit/loss metrics automatically. It’s where my love for data meets my favorite hobby. 📈

#### 🐾 [Pet Emotion Detector](https://github.com/xjiang16/Pet-Emotion-Detector)
**Stack: Python, TensorFlow/Keras, AWS, Docker**
> I built a cloud-based AI to classify pet emotions from a 16,000+ image dataset. 
> * **Model:** Leveraged **MobileNetV2** for deep learning classification.
> * **Engineering:** Solved critical GPU memory constraints by containerizing the application with **Docker** and deploying it to **AWS EC2** for real-time inference.
> * **Result:** Achieved 70% accuracy with a scalable pipeline ready for cloud or local deployment.
